Mail Triage CLI

Read-only inbox triage for fast morning scanning, sender clustering, and machine-friendly output.

View on GitHub Get Started

Scope

Uses `Mail.ReadBasic` only. No write permissions, no admin-consent creep.

Commands

  • inbox for recent message scanning
  • senders for quick sender clustering
  • --output json for automation

Auth

Built on the shared `MTG Shared Microsoft Auth` package with WAM-first sign-in on Windows.

Quick Start

$env:MAIL_TRIAGE_CLIENT_ID='e02be6f7-063a-46a6-b2cc-109d5f51055c' $env:MAIL_TRIAGE_TENANT_ID='a3599b15-c39c-4b41-a219-7e24dd5b5190' $env:MAIL_TRIAGE_SCOPES='Mail.ReadBasic' $env:MAIL_TRIAGE_AUTH_MODE='wam' $env:MAIL_TRIAGE_ALLOW_BROKER='true' .\invoke.ps1 inbox --unread-only --limit 10
